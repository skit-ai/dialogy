dialogy.plugins.text.calibration package¶

Module contents¶

This module provides a plugin to calibrate the ASR output. We look for Acoustic model and Language model scores of the ASR and predict the quality of the transcripts. Poor transcripts are filtered out. This plugin ships a trainable component.

filter_asr_output(utterances: List[List[Dict[str, Optional[Union[str, float]]]]], lang: str) → List[str][source]¶

Filter ASR output to return alternatives considered safe for classification.

The good alternatives are decided by the pWER (predicted WER). If the pWER is less than the threshold. Also returns the list of predicted wers.
