Keword Entity¶

Provides the entity class for entities generated via patterns or tokens. This entity is obtained via ListSearchPlugin and ListSearchPlugin.

class KeywordEntity(body, dim=None, parsers=NOTHING, score=None, alternative_index=None, alternative_indices=None, latent=False, values=NOTHING, value=None, entity_type=None, meta=NOTHING, *, range, type='value')[source]¶

Bases: dialogy.types.entity.base_entity.BaseEntity

Use this type for handling keyword based extractions where presence of specific tokens in the ASR is enough for detection.

alternative_index: Optional[int]¶

The index of transcript within the ASR output: List[Utterances] from which this entity was picked up. This may be None if the entity was generated from a different sources.

alternative_indices: Optional[List[int]]¶
body: str¶

The string from which the entity is extracted.

dim: Optional[str]¶

Influenced from Duckling’s convention of dimensions.

entity_type: Optional[str]¶

Describes the type of the value of an entity. Could be one of “time”, “date”, “people”, “number”, etc.

latent: bool¶

Duckling influenced attribute, True means this entity’s type wasn’t a strict match. Example: “may” is latent hint for a month in English, because it is also a modal.

parsers: List[str]¶

The trail of all the functions that have created/updated this entity at any point.

range: Dict[str, int]¶

The location of an entity marked via start and end indices on the string.

body = "i'll come by 4 pm"
entity = "4 pm"
range = {"start": 13, "end": 17}
score: Optional[float]¶

A positive real number that describes the confidence that the range contains the correct entity. A value between 0 and 1.

type: str¶

One of “value” or “interval”.

value: Any¶

A single value interpretation from values.

values: List[Dict[str, Any]]¶

Duckling stores value of an entity within a list for some types. We try to keep that as a standard for all entities.
